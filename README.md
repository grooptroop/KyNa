
# KyNa

Kyna - Платформа для автоматического развёртывания инфраструктуры по заданным параметрам

Проект предназначен для автоматизированного provision’а пользовательской инфраструктуры в Kubernetes на основе заранее заданных параметров. Backend на Go принимает запрос, сохраняет состояние в PostgreSQL и через Kubernetes API создаёт namespace, ресурсы и маршруты доступа.

---

Технологический стэк: 
- Go 
- k3s 
- ansible 
- metalLB
